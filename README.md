# FlowMaster - Kanban Board Application

A modern, feature-rich Kanban board application built with React, TypeScript, and Zustand for efficient project management and task tracking.

## 🚀 Features

### Core Functionality
- ✅ **Multi-Workflow Management**: Create and manage multiple Kanban workflows
- ✅ **Drag & Drop Interface**: Intuitive task and stage reordering
- ✅ **Real-time Updates**: Instant synchronization across the application
- ✅ **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

### Task Management
- 📝 **Task Creation & Editing**: Rich task forms with titles, descriptions, and dates
- 📅 **Date Tracking**: Start and due date management with visual indicators
- 🔄 **Task Operations**: Edit, delete, and move tasks between stages
- 🎯 **Stage Customization**: Add, edit, delete, and reorder workflow stages

### Authentication & Security
- 🔐 **User Authentication**: Secure login and registration system
- 🔑 **Session Management**: Persistent user sessions with automatic logout
- 🛡️ **Protected Routes**: Secure access to application features
- 👤 **Demo Credentials**: Easy testing with pre-configured accounts

### User Experience
- 🌙 **Dark Theme**: Modern dark UI with accent colors
- ⏳ **Loading States**: Visual feedback for all async operations
- ❌ **Error Handling**: Comprehensive error messages and validation
- ⌨️ **Keyboard Shortcuts**: Enhanced productivity with keyboard navigation

## 📦 Installation

### Prerequisites
- ![Node.js](https://img.shields.io/badge/-Node.js-339933?style=flat&logo=node.js&logoColor=white) Node.js (v16 or higher)
- ![npm](https://img.shields.io/badge/-npm-CB3837?style=flat&logo=npm&logoColor=white) npm or yarn package manager

### Quick Start

1. **Clone the repository**

git clone https://github.com/yourusername/flowmaster.git
cd flowmaster


2. **Install dependencies**

npm install


## 🎯 Usage

### Getting Started

#### 1. Authentication
- 📝 Register a new account or use demo credentials
- 🔑 **Demo**: `admin@flowmaster.com` / `password123`

#### 2. Create Your First Workflow
- ➕ Click the "+" button in the sidebar
- ✏️ Enter a workflow name and press Enter
- 🎉 Your workflow will be created with default stages

#### 3. Manage Tasks
- ➕ Click "+" on any stage to add a task
- 📝 Fill in task details (title, description, dates)
- 🔄 Use drag & drop to move tasks between stages

#### 4. Customize Stages
- ⚙️ Enable "Edit Mode" to modify stages
- 🔄 Drag stages to reorder them
- ➕ Add new stages or ❌ delete existing ones
